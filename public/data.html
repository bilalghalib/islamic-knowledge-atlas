Data Pipeline

Step 1: VIAF Extraction

Downloaded the complete VIAF 2024 clusters XML dump (21.4 GB compressed, ~150 GB uncompressed). Streamed through a Python parser that extracted pre-1500 authors with death dates, work titles, library source counts, and name variants.

Result: 85,043 author records

Key detail: The extraction streams through the gzip without decompressing to disk. Only the matching records are kept.

Step 2: AI Classification

Each of the 36,158 scholars with work titles was classified individually by Claude Sonnet 4.6, producing:

FieldDescription
title_evidenceDomains supported ONLY by the catalog titles. Each entry: domain, evidence quote, confidence (0-100)
background_knowledgeDomains the AI knows about beyond the titles. Each entry: domain, explanation, confidence, Wikipedia reference
chaney_strictTrue if title evidence alone supports a science domain with confidence ≥ 60
chaney_broadTrue if any evidence supports a science domain with confidence ≥ 40
polymath_strict/broadTrue if 3+ distinct domains at respective confidence thresholds
regionBest estimate: Mashriq, Maghrib, Andalus, Central_Asia, South_Asia, Europe, East_Asia

Key design choice: Two classification layers (title vs background) so you can distinguish "what the data shows" from "what we know." The gap between them IS the argument: library catalogs can't see what the tradition knows.

Step 3: Recovery

The classification database was accidentally dropped during a script crash. We recovered 35,606 of 35,978 records (99.0%) from SQLite's freelist pages, then matched them back to VIAF records using title evidence strings and Wikipedia URLs as matching keys.

Verification rate: 78.6% matched by title content, remainder matched by sequential ordering with fuzzy verification.

Note: falsafa (philosophy) and mantiq (logic) are classified as "Islamic Sciences" following the tradition's own categorisation. Chaney would not count these as science. This is itself a finding: the tradition classified rational inquiry as religious duty.

Limitations

  • VIAF overrepresents scholars whose works survived in modern Western library collections (83% Western library coverage)
  • AI classification depends on transliterated titles, which vary across library systems
  • Background knowledge may introduce post-hoc information not available to contemporaries
  • Region assignment is approximate (based on AI inference from names and titles)
  • Confidence scores are the AI's self-assessment, not independently validated
  • The 21.4% of records that couldn't be matched during recovery may have slightly different characteristics than the matched records
